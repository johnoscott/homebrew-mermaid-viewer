# homebrew-mermaid-viewer

Homebrew tap for [MermaidViewer](https://github.com/johnoscott/MermaidViewer) â€” a macOS Quick Look extension for previewing Mermaid diagram files (.mmd, .mermaid) in Finder.

## Install

```bash
brew install johnoscott/mermaid-viewer/mermaid-viewer
```

Or add the tap first:

```bash
brew tap johnoscott/mermaid-viewer
brew install mermaid-viewer
```

## Update

```bash
brew upgrade mermaid-viewer
```

## Uninstall

```bash
brew uninstall mermaid-viewer
```
